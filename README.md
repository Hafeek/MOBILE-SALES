# Motorola Mobile Sales Analytics Dashboard | Power BI

### 📌 Problem Statement
Analyzed Motorola mobile sales data to track revenue, customer behavior, and payment trends across North India for a simulated retail client.

### 🛠️ Tools Used
- **Power BI:** Data visualization, DAX, Maps, Slicers
- **Excel:** Data cleaning, sample dataset creation
- **Domain:** Retail Analytics, Sales KPI Tracking, Customer Segmentation

### 📊 Key Metrics & Insights

#### **KPI Summary**
| Metric | Value | Business Meaning |
| --- | --- | --- |
| Total Sales | ₹550K | Overall revenue from mobile sales |
| Total Quantity | 19 Units | Total mobiles sold |
| Transaction Count | 12K | *Note: Data mismatch - needs validation* |
| Top City | Delhi/Jaipur | Highest sales region in North India |

#### **Critical Business Insights**
1. **Customer Concentration:** Sumit Verma contributed 42.11% of total units sold = 8 mobiles  
   → **Action:** Launch VIP loyalty program for high-value customers.

2. **Geography:** 100% sales from North India - Delhi, Jaipur, Jodhpur belt  
   → **Action:** Focus marketing spend in these cities. Expand to Tier-2 next.

3. **Payment Mix:** UPI/Cash/Credit each 33% share  
   → **Action:** UPI adoption healthy. Promote credit card EMI for higher ASP.

4. **Brand Mix Issue:** Dataset shows Apple/Samsung/Vivo sales under Motorola dashboard  
   → **Action:** Indicates multi-brand retailer data. Segment Motorola vs competitor sales for accurate brand analysis.

### 🎯 Recommendations for Sales Team
1. **For Top Customer:** Give Sumit Verma early access to new Motorola launches.
2. **For North India:** Double dealer count in Jaipur/Jodhpur during festive season.
3. **For Payments:** Run 'No Cost EMI on Credit Card' campaign to increase ASP.

### ⚠️ Data Quality Observations
This dashboard uses sample data of 3 customers. Found inconsistencies: 
1. `Sum of TRANSACTION = 12K` vs `Total Quantity = 19` - needs data validation
2. Brand filter shows Apple/Samsung under Motorola dashboard
3. World map used for India-only data
Demonstrates ability to audit dashboards for accuracy before presenting to stakeholders.
